Public API

Alternatively, instead of using an adapter, you can use so called public API to access a controller method from outside the controller. The only difference between an adapter and a public API is that adapter exposes every single controller method to the outside world, whereas public API only exposes those controller methods which are marked as a part of the public API in the controller. To mark a controller method as a part of the public API, use the @PublicApi annotation. Instead of defining the method as a field of the controller, define a controller field called "@PublicApi" instead and put all publicly accessible methods as its fields, as the following example demonstrates.
