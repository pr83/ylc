Controller mixins

Yellow code allows you to reuse controller code via the mixin concept. If multiple Yellow code controllers in your project contain the same methods (be it event handlers or custom functions), you may wish to factor the common pieces out into a separate object. Such an object is called a mixin.

There are two ways how to make the mixin methods accessible to a Yellow code view:

  • Locally for a single view - when binding the view and the controller via $(...).yellowCode(...), instead of passing a single controller to Yellow code, pass an array where the first element is the controller and the remaining elements are one or more mixins:
    $("#viewId").yellowCode([controller, mixin1, mixin2, ...]);

  • Globally for the whole page - this will make the mixin automatically accessible to all views:
    $.yellowCode.standardMixins = [mixin1, mixin2, ...];

The effect of making the mixins accessible to the view is the same as if you copy-pasted the methods of all the mixins and appended them to the existing methods of the controller.
